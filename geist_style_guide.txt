
GEIST WEB APP — VISUAL DESIGN GUIDE
====================================

THEME:
------
Spectral, glitch-punk, cyber-alternative. Inspired by underground queer rave culture, squatter aesthetics, and dreamlike UI styles from sites like namedcollective.com, dice.fm, and dropdead.world.

COLORS:
-------
- Primary Background: #0c0c0c (Deep Black)
- Foreground Text: #f4f4f4 (Bone White)
- Accent Purple: #8a2be2 (Geist Violet)
- Highlight Pink: #ff00ff (UV Pink Glitch)
- Glitch Cyan: #00ffff
- Muted Grey: #666 (Ghost Ash)

TYPOGRAPHY:
-----------
- Main Font: 'Inter', sans-serif (clean, sharp)
- Logo / Headers: Custom Geist font (from Nameless Entities logo lettering)
- Uppercase for main titles
- Soft spacing, tight kerning

LAYOUT:
-------
- Full bleed background
- Fixed bottom tab navigation bar
- Containers with rounded edges and inner glow
- Scrollable chat + log area with soft borders

BUTTONS:
--------
- Bold, rectangular with glitch-hover effect
- Glow accent on hover
- Secondary buttons outlined in glitch cyan or pink

EFFECTS:
--------
- Glitch animation (distortion-style on hover/headings)
- Subtle SVG overlays (dust, grain, tribal marks)
- Background movement (slow pulse or float)

ICONS:
------
- Custom icons from `/assets/`
- Use tribal linework and spectral shapes
- Tabs can be symbol-based (e.g. skull for health, flame for fitness)

NAVIGATION:
-----------
- Tabs at bottom for mobile-first experience
- Ghost glow active state
- Dynamic switch between content sections using smooth transitions

INTEGRATIONS:
-------------
- Spectral splash screen from `/assets/splash/`
- Geist tribal logo always visible top-center on load
- Chat window styled like a ritual interface

ANIMATION IDEAS:
----------------
- Logo flickers on first load
- Glitch-pulse buttons
- Chat input “glows” when focused
- Loading spinner made from tribal glyphs
